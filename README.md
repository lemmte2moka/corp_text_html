### npm installでパッケージをインストール
```
npm install
```

## gulpの構成
本テンプレートは開発用フォルダで作成/追加したファイル郡を、公開用フォルダに出力するように作っています。  

|ディレクトリ|用途|
|-----|-----|
|src|開発用ディレクトリ。新しくファイルやディレクトリを追加する時は`src`に追加してください。|,
|dist|公開用ディレクトリ。サーバーアップする時や納品する時のディレクトリです。

---

### タスクの使い分け

##### 開発時
```
npm start
```

##### サーバアップ/納品時
```
npm run build
```

# ディレクトリ構成

- src/
  -  assets/ （資材格納ディレクトリ、他ディレクトリでも動作可能）
     -  css/
     -  fonts/
     -  img/
     -  include/
     -  js/ 
  - index.html
- dist/ （ファイル出力先ディレクトリ）
